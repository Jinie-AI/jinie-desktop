# Live Review Panel Component (Module 14)

Provides sandboxed web visualization previews for generated apps and components.

## Submodules / Features

### 1. Component Preview (`ComponentPreview.tsx`)
Inspects generated design assets (cards, banners, buttons) in isolation.

### 2. Full App Preview (`FullAppPreview.tsx`)
A simulated browser device viewport showing sitemap transitions and interactions.

### 3. Device Toggle (`DeviceToggle.tsx`)
Switches layout breakpoints (Desktop, Tablet, Mobile) to review response sizing.

### 4. Feedback Capture (`FeedbackCapture.tsx`)
Lets the user highlight elements to post correction comments back to the engine.
