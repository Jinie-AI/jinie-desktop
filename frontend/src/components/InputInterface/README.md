# Input Interface Component (Module 10)

Captures the user's initial application design concepts in English or Urdu.

## Submodules / Features

### 1. Prompt Input Box (`PromptInputBox.tsx`)
A text input area supporting plain text and mixed language inputs to specify what kind of e-commerce application is desired.

### 2. File Uploader (`FileUploader.tsx`)
Allows attachments like reference designs, site maps, or images to assist in generation context.
