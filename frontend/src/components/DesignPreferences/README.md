# Design Preferences Panel Component (Module 11)

Collects the user's styling decisions and parameters.

## Submodules / Features

### 1. Color Palette (`ColorPalette.tsx`)
Enables choosing primary, secondary, and accent colors.

### 2. Typography (`Typography.tsx`)
Lets the user select fonts for headings and body copies.

### 3. Layout Mode (`LayoutMode.tsx`)
A selector for navigation setups (tabs, drawer, custom).

### 4. Theme Toggle (`ThemeToggle.tsx`)
Configures dark/light themes.
